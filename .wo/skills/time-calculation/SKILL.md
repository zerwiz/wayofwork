## Construction time calculation (Tidsplanering)

### Time per unit standards

Swedish construction uses standardised time estimates per work type, typically expressed in **timmar per enhet** (hours per unit):

| Work type | Unit | Standard time | Notes |
|---|---|---|---|
| Formwork (formbyggnad) | tim/m² | 1.5–3.0 | Depends on complexity |
| Reinforcement (armering) | tim/ton | 8–14 | +20% for complex beams |
| Concrete casting (gjutning) | tim/m³ | 0.5–1.5 | Including finishing |
| Masonry (murning) | tim/m² | 1.0–2.5 | Per 250 mm block |
| Carpentry (snickeri) | tim/m² | 0.8–2.0 | Interior walls |
| Flooring (golvläggning) | tim/m² | 0.3–0.8 | Laminate/parquet |
| Painting (målning) | tim/m² | 0.2–0.5 | Per coat |
| Ground work (markarbete) | tim/m³ | 0.1–0.4 | Machine-assisted |
| Plumbing (VS-montage) | tim/enhet | 2–6 | Per fixture point |
| Electrical (el-montage) | tim/enhet | 1–4 | Per outlet/switch |

### Crew composition and productivity

| Crew type | Typical size | Productivity factor |
|---|---|---|
| Snickarlag (carpenter crew) | 2–3 pers | 1.0 (baseline) |
| Betonglag (concrete crew) | 3–5 pers | 1.2 (coordination overhead) |
| Marklag (ground crew) | 2–4 pers | 0.9 (machine multiplier) |
| Installationslag (MEP crew) | 2–3 pers | 1.1 (cramped space factor) |
| Stomlag (frame crew) | 4–6 pers | 1.3 (safety overhead) |

### Work phase time distribution (typical building project)

1. **Grundläggning (Foundation)** — 10-15% of total time
   - Schakt, pålning, gjutning
2. **Stomme (Frame)** — 25-35%
   - Betong eller stål/stomkomplettering
3. **Tak och fasad (Roof & Facade)** — 10-15%
4. **Installationer (MEP)** — 15-20%
   - Värme, vatten, el, ventilation
5. **Innergolv och ytskikt (Interior)** — 15-20%
   - Väggar, golv, målning, kakel
6. **Slutstädning och besiktning (Finish)** — 5-10%

### Dependencies and critical path

- Foundation → Frame → Weathertight → Interior → Finish
- MEP installations typically run parallel with interior phase
- Torktid (drying time) for cast concrete: min 1 day per 10 mm thickness
